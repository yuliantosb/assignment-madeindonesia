# Next.js App Setup & Installation Guide

This guide explains how to install, run, and deploy a Next.js project from scratch.

---

## 1. Prerequisites

Make sure you have installed:

- Node.js (v18+ or v20 recommended)
- npm / yarn / pnpm
- Git

Check versions:

```bash
node -v
npm -v
git --version
```

## 2. Clone the Project

```bash
git clone https://github.com/yuliantosb/assignment-madeindonesia.git
cd assignment-madeindonesia
```

## 3. Install Dependencies

Choose one package manager:

```bash
npm install
```

yarn

```bash
yarn install
```

pnpm

```bash
pnpm install
```

## 4. Run Development Server

```bash
npm run dev
```

Open in browser

```bash
http://localhost:3000
```
